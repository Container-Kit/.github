# 👋 Welcome to ContainerKit!

We are dedicated to building high-performance, developer-first tools that bridge the gap between complex system utilities and intuitive user interfaces. 
**ContainerKit** is a GUI application designed specifically for **Apple’s native Container CLI** (introduced in 2025).

By leveraging the power of macOS-native containerization, we provide a streamlined management experience that doesn't compromise on speed or system resources.

---

### 🚀 Why ContainerKit?

Most container management tools are heavy and cross-platform by necessity. We’ve taken a different approach: **Native First.**

* **Optimized Performance:** Built with **Rust** and **Svelte 5** to ensure a minimal resource footprint and lightning-fast startup times.
* **macOS Integration:** Deeply integrated with Apple's virtualization and container frameworks for a seamless experience.
* **Security & Transparency:** We utilize advanced OCI metadata analysis and integrated security scanning to keep your development environment safe. (Have you heard of Trivy?)
* **Lightweight Design:** We aim for highly optimized binaries, ensuring your workflow stays agile.

### 🛠 Tech Stack

Our engineering philosophy focuses on modern, type-safe, and compiled languages to deliver the best possible desktop performance.

| Layer | Technology |
| :--- | :--- |
| **Frontend** | Svelte 5 (SvelteKit) |
| **Backend** | Rust |
| **Framework** | Tauri |
| **Interface** | Apple Container CLI (Native) |
